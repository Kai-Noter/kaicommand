# How to Add Topics to Your GitHub Repo

Topics are short labels that help people find your project on GitHub (like hashtags).

---

## Steps

1. **Open your repo**  
   Go to: `https://github.com/jackwinga-cyber/KaiCommand`

2. **Find the About box**  
   On the **right side** of the page, look for the **About** section (under the Watch / Fork / Star buttons).

3. **Click the gear icon**  
   Next to "About" there is a small **gear (⚙️)** or **pencil** icon. Click it.

4. **Add topics**  
   In the **Topics** field, type a topic and press **Enter** to add it. Add as many as you like. Suggested topics for KaiCommand:

   - `nextjs`
   - `typescript`
   - `react`
   - `ai`
   - `productivity`
   - `prisma`
   - `tailwindcss`
   - `next-auth`

5. **Save**  
   Click **Save changes** (or just click outside the box). The topics will appear as grey pills under the description.
